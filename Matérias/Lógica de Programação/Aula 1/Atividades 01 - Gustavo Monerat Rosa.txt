A) Pesquise e responda:

1) O que é algoritmo?
Conjunto de regras finitas para solucionar um problema.

2) O que é um programa de computador?
Conjunto de instruções para solucionar um problema.

3) O que é uma variável?
É um espaço na memoria onde pode ser armazenada e atualizada a informação.

4) O que é o Portugol Studio?
Um ambiente de desenvolvimento (IDE) para a linguagem Portugol.

5) O que faz o comando escreva do Portugol?
Imprime no console oque você indicar dentro dessa função escreva. Ou seja uma função que gera um output.

6) O que faz o comando leia do Portugol?
Pede ao usuario que insira uma informação. Ou seja uma função que gera input.

7) Defina o que é tipo de variável
O tipo de variável é a categoria de dados que ela armazena.

8) Explique o que são os tipos de variáveis do Portugol:
   a. cadeia -> String = cadeia de caracteres ("Texto")
   b. inteiro -> Int = Numeros inteiros (1,2,-3)
   c. real -> Float = Numeros racionais (1.2,1.3,-1.0)
   d. logico -> Booleano = Verdadeiro ou Falso (verdadeiro)
   e. caracter -> Char = Unico caractere ('c','g','a')

9) Dê exemplo:
   a. de entrada de um algoritmo
	leia(nota1)
	leia(nota2)
   b. de processamento de um algoritmo
	media = (nota1+nota2)/2
   c. de saída de um algoritmo
	escreva("A média é: ", media)
